CLT-LARW服务 文档v1.0
介绍
CLT-LARW服务是建立在Salesforce.com Live Agent和Wechat之上的服务，即在线客服系统。 使用本服务，不需要您或你的组织进行任何开发，仅需将您在线客服相关信息（Live Agent & Wechat）录入系统即可使用， 但您的额外业务需求不在此范围内。本服务采用标准Salesforce API和Wechat API，独立运行。
配置
1.点击进入注册页面，配置您或您组织的相关内容后，即可使用；
2.点击进入Live Agent 配置文档；
3.点击进入Wechat 配置文档；
说明
1.若您浏览了上述配置文档，您会意识本服务的便捷性，仅仅需要数分钟的信息录入即可使用；
2.本服务会在后续进行陆续升级，届时会发送邮件给您。如果您在使用本服务，请确保您的邮箱正确方便接收邮件以免遗漏信息；
3.欢迎您提出任何意见和建议，您的留言将得到我们真诚的处理，也期待本服务带给您的便捷。
限制
1.在v1.0中，您或您组织的服务将被限制在每个org，允许一个配置、一个按钮；
2.在v1.0中，您或您组织的服务连接个数初始将被限制在3个连接，若您有需求请及时联系我们。我们期望让更多的用户得到体验资格，尽请谅解；
3.Wechat定义了单向到Wechat User的信息条数，故请注意在Chat过程中，单向信息的限制；
4.在v1.0中，提供两种刷新Wechat access_token方式：本服务代理，或您定时向本服务推送。本服务代理，暂不支持向外输出access_token,刷新时间为90 - 100分钟一次；若您推送，请注意推送时间，以免导致无法连接Wechat服务。
样例
1.点击进入Salesforce 后台；
2.使用Wechat扫描下方二维码，关注该公众号；

3.使用流程:
1）.在Salesforce Console底部右侧选择 Live Agent 标签，保持在线状态；
2）.在该公众号内发送消息（若您未保持Live Agent在线，将收到默认无客服消息）；
3）.推荐您使用您或您组织的Org、Sandbox体验本服务。
愿景
1.我们期待本服务会给您带来真正的便捷，实现您或您组织的业务快速更迭；
2.欢迎您的使用！我们会在后面陆续推出各类型的服务，敬请关注。
