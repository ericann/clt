微信设置 文档
登录
1.登录微信公众号官网 点击登录
Account Id
找到左侧导航栏 “设置” 下的 “公众号设置”，点击进入，在主面板最下方可以看到 “注册信息” 下的 “原始ID”，即为本服务所指Account Id。如下图：
App Id/App Secret/Token
找到导航栏 “设置” 下方 的 “开发” - “基本配置”， 点击进入，在主面板上方可以看到：
1、 “开发者ID” 即本服务所指App Id；
2、 “开发者密码” 即本服务所指App Secret；
3、 “令牌” 即本服务所指Token。若需本服务刷新Access Token，请在 “服务器地址” 内输入：http://47.93.254.177/clt/wechat/msg/text（“令牌” 自定义填写）。如下图：
注
在微信公众号更新后，刷新Access Token, 需将刷新服务器置为IP白名单。因此，若您需要本服务刷新Access Token，请将IP：47.93.254.177，设置在IP白名单内；否则，请您及时将Access Token推送到本服务中。
推送地址：http://47.93.254.177/clt/token/refresh/single；
推送类型：POST；
数据类型：JSON；
数据样板：
{
    wechatAccount: "Your Account Id",
    access_token: "refreshed access token"
}
