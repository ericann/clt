Salesforce.com Live Agent设置 文档
前置步骤
1.登录Salesforce Org. Pro/Developer Sandbox
2.点击设置，进入设置界面
Organization ID
在左侧Quick Box内输入 “公司简档” ，点击公司信息后，在主面板最下方可以看到 “Salesforce.com 组织 ID”，即为本服务所指Organization ID。若您是开发者，请直接从数据库查询。如下图：
Deployment ID
在左侧Quick Box内输入 “live agent”， 点击 “部署”， 在主面板可以看到部署列表，选择要使用的部署，点击进入，截取浏览器地址栏后的15位字符。若您是开发者，请直接从数据库查询。如下图：
Button ID
在左侧Quick Box内输入 “live agent”， 点击 “聊天按钮 & 邀请”， 在主面板可以看到按钮列表，选择要使用的按钮，点击进入，截取浏览器地址栏后的15位字符。若您是开发者，请直接从数据库查询。如下图：
End Point
在左侧Quick Box内输入 “live agent”， 点击 “Live Agent 设置”， 在主面板可以看到 “API 信息” - “Live Agent API 端点”，复制输入框内的地址即可。如下图：
注
1.在当前版本中，仅支持单个配置下的单个按钮，在后续版本中将陆续开放多按钮、多配置。
2.在截取字符时，请注意是15位，若您从数据库查询请截断后三位。
